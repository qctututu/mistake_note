# 高一数学 · 集合 课堂笔记

## 一、集合的基本概念

### 1.1 什么是集合

**定义：** 把一些确定的、互不相同的对象看成一个整体，这个整体就是集合。集合中的每个对象称为**元素**。

**三个特性：**
- **确定性：** 元素要么属于要么不属于，不能模棱两可
- **互异性：** 集合中的元素互不相同，重复的只算一个
- **无序性：** 集合中的元素不考虑顺序

### 1.2 元素与集合的关系

| 符号 | 含义 | 示例 |
|------|------|------|
| $a \in A$ | a 属于集合 A | $1 \in \{1,2,3\}$ |
| $a \notin A$ | a 不属于集合 A | $4 \notin \{1,2,3\}$ |

### 1.3 常用数集

| 符号 | 含义 |
|------|------|
| $\mathbb{N}$ | 自然数集（含 0） |
| $\mathbb{N}^*$ 或 $\mathbb{N}_+$ | 正整数集 |
| $\mathbb{Z}$ | 整数集 |
| $\mathbb{Q}$ | 有理数集 |
| $\mathbb{R}$ | 实数集 |
| $\emptyset$ | 空集（不含任何元素） |

### 1.4 集合的表示法

**列举法：** 把元素一一列出，用 `{}` 括起来
```
A = {1, 2, 3, 4, 5}
B = {a, b, c, d}
```

**描述法：** 用元素满足的条件来描述
```
A = {x | x ∈ ℕ, x < 6}
B = {x | -2 ≤ x < 3, x ∈ ℤ}
```

**Venn 图：** 用封闭曲线表示集合（画图时可直观理解）

## 二、集合间的基本关系

### 2.1 子集

**定义：** 若集合 A 的**每一个**元素都属于集合 B，则称 A 是 B 的子集

符号：$A \subseteq B$ 或 $B \supseteq A$

**性质：**
- 任何集合都是自身的子集：$A \subseteq A$
- 空集是任何集合的子集：$\emptyset \subseteq A$

### 2.2 真子集

**定义：** 若 $A \subseteq B$ 且 $A \neq B$，则称 A 是 B 的真子集

符号：$A \subsetneq B$

### 2.3 集合相等

**定义：** 若 $A \subseteq B$ 且 $B \subseteq A$，则 $A = B$

**证明方法：** 双向包含法——先证 A 中任意元素都在 B 中，再证 B 中任意元素都在 A 中

### 2.4 子集个数

若集合 A 有 n 个元素，则：
- 子集个数：$2^n$
- 真子集个数：$2^n - 1$
- 非空真子集个数：$2^n - 2$

**例题：** 若 $A = {a, b, c}$，求 A 的子集个数
解：n = 3，子集个数 = $2^3 = 8$ 个

## 三、集合的运算

### 3.1 交集

**定义：** 所有既属于 A 又属于 B 的元素组成的集合

符号：$A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$

**性质：**
- $A \cap A = A$
- $A \cap \emptyset = \emptyset$
- $A \cap B \subseteq A$，$A \cap B \subseteq B$
- 交换律：$A \cap B = B \cap A$
- 结合律：$(A \cap B) \cap C = A \cap (B \cap C)$

### 3.2 并集

**定义：** 所有属于 A 或者属于 B 的元素组成的集合

符号：$A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$

**性质：**
- $A \cup A = A$
- $A \cup \emptyset = A$
- $A \subseteq A \cup B$，$B \subseteq A \cup B$
- 交换律：$A \cup B = B \cup A$
- 结合律：$(A \cup B) \cup C = A \cup (B \cup C)$

### 3.3 补集

**全集：** 用 $U$ 表示，包含所有讨论对象的集合

**补集：** 由全集 U 中**不属于** A 的所有元素组成的集合

符号：$\complement_U A = \{x \mid x \in U \text{ 且 } x \notin A\}$

**性质（德摩根律）：**
- $\complement_U (A \cap B) = (\complement_U A) \cup (\complement_U B)$
- $\complement_U (A \cup B) = (\complement_U A) \cap (\complement_U B)$

### 3.4 德摩根律记忆技巧

> "交之补等于补之并，并之补等于补之交"
> 符号记忆：长括号变短运算，交变并、并变交

## 四、典型题型

### 题型 1：元素互异性

**例题：** 若集合 $A = \{x, x^2, 2\}$ 中有 3 个不同的元素，求 x 的取值范围

**分析：** 互异性要求 $x \neq x^2$，$x \neq 2$，$x^2 \neq 2$

**解：**
- 由 $x \neq x^2$ 得 $x \neq 0$ 且 $x \neq 1$
- 由 $x \neq 2$ 得 $x \neq 2$
- 由 $x^2 \neq 2$ 得 $x \neq \pm \sqrt{2}$

综合得：$x \in \mathbb{R}$ 且 $x \neq 0, 1, 2, \sqrt{2}, -\sqrt{2}$

### 题型 2：子集与真子集

**例题：** 已知集合 $A = \{x \mid x^2 - 3x + 2 = 0\}$，写出 A 的所有子集

**解：** $x^2 - 3x + 2 = 0 \Rightarrow (x-1)(x-2) = 0$
$\therefore A = \{1, 2\}$
子集：$\emptyset$，$\{1\}$，$\{2\}$，$\{1, 2\}$（共 $2^2 = 4$ 个）

### 题型 3：集合运算

**例题：** 设 $U = \{1,2,3,4,5\}$，$A = \{1,2,3\}$，$B = \{2,3,4\}$，求 $A \cap B$，$A \cup B$，$\complement_U A$

**解：**
- $A \cap B = \{2, 3\}$（公共元素）
- $A \cup B = \{1, 2, 3, 4\}$（所有元素，去重）
- $\complement_U A = \{4, 5\}$（U 中不在 A 里的）

### 题型 4：含参集合

**例题：** 已知 $A = \{x \mid x^2 - ax + 6 = 0\}$，且 $1 \in A$，求实数 a 及 A 中所有元素

**解：** $1 \in A \Rightarrow 1 - a + 6 = 0 \Rightarrow a = 7$
方程：$x^2 - 7x + 6 = 0 \Rightarrow (x-1)(x-6) = 0$
$\therefore A = \{1, 6\}$

### 题型 5：集合应用题

**例题：** 高一某班 50 人，喜欢数学的 30 人，喜欢物理的 25 人，两科都喜欢的有 15 人。求：
（1）只喜欢数学的人数
（2）两科都不喜欢的人数

**解：**
（1）只喜欢数学 = $30 - 15 = 15$ 人
（2）至少喜欢一科 = $30 + 25 - 15 = 40$ 人
两科都不喜欢 = $50 - 40 = 10$ 人

## 五、易错点总结

1. **空集 $\emptyset$：** $\emptyset \subseteq A$ 恒成立，但 $\emptyset \in A$ 不一定成立
2. **区分 $\subseteq$ 和 $\in$：** 一个表示子集关系，一个表示元素归属
3. **列举法不要漏写大括号：** $\{1,2\}$ 是集合，$1,2$ 只是两个数
4. **描述法的竖线：** $\{x \mid \text{条件}\}$，竖线不能漏
5. **补集必须先有全集：** 没有全集 U 就没有补集
6. **互异性验证：** 解出含参集合后，一定要代回去检查元素是否互异
